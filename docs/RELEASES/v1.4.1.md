# U4Claude v1.4.1

Quick-wins polish release. No new features — three small improvements bundled together.

## What changed

- **Ring illumination slider.** New "Popover Appearance" card in General Settings exposes a 0–100% slider that scales the glow around the popover rings. Drag it down to "subtle" or off entirely if the v1.3.1 glass-tube halo is too vivid for your wallpaper, or leave it at 100% (the default) for the unchanged v1.4.0 look. Above 50% the macOS-26 Liquid Glass material is layered in; below it the slider gates the material off and just dims the surrounding shadows.
- **Auto-relogin recovers from a dismissed login window.** The session-expired throttle previously latched on the first `.sessionExpired` and only released after a successful fetch — which couldn't happen without re-authenticating. If you dismissed the WebLogin window without logging in, the popup never came back. The throttle now clears on explicit user retry (manual refresh, or opening the popover after >30s), so the next failure re-prompts.
- **Spoofed Chrome user-agent bumped 140 → 148.** Real Chrome has marched on; Cloudflare's heuristics flag stale UAs more aggressively than they used to. Internal-only — nothing user-visible.

## Internal changes

- `UserSettings.ringIlluminationLevel: Double` (default 1.0), persisted in UserDefaults.
- `UsageDetailView` now `@ObservedObject`s `UserSettings.shared` so the slider live-previews while you drag it. Shadow opacity/radius scale linearly with the value; `.glassEffect(in:)` is gated above the 0.5 threshold via a private `ringGlass(when:in:)` helper.
- `DataRefreshManager.handleManualRefresh` and `refreshOnPopoverOpen` clear `sessionExpiredPrompted` before fetching.
- `ClaudeAPIHeaderBuilder` user-agent bumped.
- Localization strings added in all five locales (en, ja, ko, zh-Hans, zh-Hant).
